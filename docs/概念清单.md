**版本**：2026-05-22

> **填法**：对照下表，把题目对应行的 **推荐 category** 与 **推荐 knowledge_point** 抄到 xlsx 的对应列即可。如需用别名,从「可用别名」列里挑也行——以下任一组合都能被系统识别。
>
> 找不到合适的归类？联系项目后端同学(我),先看是否需要新增 concept,而不是凑一个最像的填进去。

## 五、LaTeX 录入约定

`content`、`options`、`explanation` 三列允许 LaTeX 原文，例如：

- 行内公式：`$a_1 = 2$`、`$\sqrt{x^2+1}$`
- 块公式：`$$\sum_{i=1}^{n} a_i = \frac{n(a_1 + a_n)}{2}$$`
- 不等式、希腊字母、上下标都可放心使用

数据库按原文存储（不转义）；前端将用 KaTeX 渲染。

## 六、常见问答

**Q: 我有一道题跨两个知识点，怎么办？**
A: 选**最核心**的那个填即可（demo 阶段不支持一题多 concept），可在 `tags` 列写上次要标签如 `跨章节,与XX相关`。

**Q: 模板的示例行(第 3 行)要不要删？**
A: 不删也可以，导入脚本会按 `(stem, source, year)` 去重，示例题被导入一次也不影响。

**Q: 题录到一半发现别名表里没有合适的怎么办？**
A: 暂时挑一个最近的填，告诉我（项目后端）哪些命名要新增，我会在 concepts seed 里加进去并重 seed。
