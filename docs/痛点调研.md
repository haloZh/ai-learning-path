# 管综 + 英语二 备考痛点调研

> **来源**：项目组成员面向在职考研群体（MBA / MPAcc / MEM 等）整理的痛点清单，用作本项目"需求调研"原始素材。
> **用途**：调研报告（10 分交付物）引用来源。Demo 首期仍按 proposal 锁定在 **管综 · 数学基础** 子科目；其他三科痛点用于证明市场共性、拓展可解释性。

---

## 一、数学（管综 · 数学基础）

### 1. 基础硬伤
1. 离校多年，公式全忘，初高中知识断层。
2. 计算能力退化，列式对、算错多。
3. 只会死算，无巧解思维，做题慢。

### 2. 考试题型痛点
1. 全部选择题 + 条件充分判断，题型特殊，新手完全摸不透。
2. 条件充分判断是最大坑，逻辑绕、极易混淆，正确率极低。
3. 题量极大，60 分钟 25 题,速度 = 生死线。

### 3. 知识高频难点(必考拉分)
1. **算术**：整除、余数、比例、绝对值,陷阱多。
2. **代数**：二次函数、不等式、数列，变形灵活。
3. **几何**：平面几何公式杂、立体空间弱、解析几何计算繁。
4. **排列组合概率**：在职头号难点，分不清分类分步，极易乱套。
5. **应用题**：行程、工程、浓度、利润，读题绕，不会建模。

### 4. 在职专属痛点
1. 碎片学习,数学需要连续思考,一打断思路清零。
2. 白天用脑透支,晚上算题反应慢、犯困卡顿。
3. 只会刷题不总结,同类题反复错,提分停滞。
4. 时间不够,没时间深耕,只能浅记,一变题型就不会。

### 5. 底层致命问题
1. 没有解题套路,全靠硬算,耗时翻倍。
2. 畏难心理,排列组合、几何直接放弃,直接丢分。
3. 粗心通病,看错条件、漏符号,失分惨重。

## 二、逻辑（管综 · 逻辑推理）

### 1. 先天痛点
1. 零基础零接触,从没学过,完全无思维框架。
2. 文字巨多、题干绕,读题慢、抓不住重点。
3. 日常思维和考试逻辑相反,自带主观脑补,必错。

### 2. 三大题型硬核难点

**(1) 形式逻辑**
1. 假言、联言、选言转换绕,推不出链条。
2. 肯后否前、偷换概念,高频踩坑。
3. 公式记混,符号转换混乱。

**(2) 论证逻辑（最大失分点）**
1. 削弱/加强/假设分不清判定标准。
2. 因果倒置、另有他因、无关选项极易看错。
3. 凭语感做题,选项迷惑性极强,越读越乱。

**(3) 综合推理（在职噩梦）**
1. 排序、分组、匹配,条件多、链条长。
2. 耗时长,一道题占用 3–5 分钟。
3. 多条件嵌套,断点就全盘推不出。

### 3. 考试致命痛点
1. 60 分钟 30 题,时间极度压缩,推理来不及。
2. 题干长篇阅读,下班脑子迟钝,反应变慢。
3. 题干文字晦涩,信息杂,筛选能力弱。

### 4. 在职专属短板
1. 碎片化学习,逻辑需要连贯推演,中断重来全乱。
2. 不会标准化套路,全靠硬读硬想。
3. 刷题不复盘,错题只记答案,不会底层规则。

### 5. 核心根源
逻辑不靠聪明,难在：抛弃日常语感 + 固定推理模板 + 极速读题。

## 三、英语二

### 1. 词汇痛点
1. 离校多年,单词全忘,零基础重启。
2. 熟词僻义多,认识本义,考题释义看不懂。
3. 背了就忘,碎片记忆,复盘跟不上。
4. 考研一词多义,日常词义没用,考场上全错。

### 2. 阅读最大拉分坑
1. 长难句看不懂,句子长、从句嵌套,拆分不开。
2. 逐词认识,连起来读不通。
3. 中式思维,容易主观脑补,掉入选项陷阱。
4. 阅读量大,做题慢,没时间细读。

### 3. 语法硬伤
1. 时态、从句、非谓语完全混乱。
2. 不会分析句子结构,只会死翻译。
3. 基础语法断层,写作翻译全写错。

### 4. 完形 + 新题型
1. 完形近义词辨析弱,逻辑连接词看不懂。
2. 新题型行文逻辑松散,抓不住段落关系。

### 5. 翻译写作痛点
1. 翻译语序混乱,中式直译,不通顺。
2. 写作无模板、无句型,只会简单短句。
3. 语法错误满天飞,拼写差、时态乱。
4. 无话可写,凑词生硬。

### 6. 在职专属致命问题
1. 下班脑子疲惫,看英文直接走神犯困。
2. 没时间长期积累,只能突击,根基浅。
3. 记词断断续续,越背越混乱。

## 四、写作（管综 · 中文写作）

### 1. 论证有效性分析
1. 不懂逻辑谬误,找不到漏洞。
2. 只会直白反驳,不会专业话术。
3. 只会套模板,分析很浅。
4. 混淆归因、以偏概全、偷换概念。
5. 字数不够、语句啰嗦,结构松散。

### 2. 论说文
1. 审题易偏,立意浅薄。
2. 无素材、无论据,只会空喊。
3. 不会分层,段落杂乱无框架。
4. 论证断层,观点和例子不贴合。
5. 文笔退化,通篇大白话。

### 3. 考场硬难点
1. 55 分钟写两篇,时间极度紧张。
2. 字迹潦草,书写速度慢。
3. 紧张焦虑,思维卡顿写不出。

### 4. 在职专属痛点
1. 常年不动笔,提笔无话。
2. 下班脑力枯竭,思辨力下降。
3. 碎片化学习,练得少、复盘少。
4. 不会精简模版,考前只会硬凑。

## 五、四科共性洞察（服务于产品定位）

| 共性痛点 | 出现科目 | 对产品的指引 |
|---------|---------|-------------|
| **碎片化学习，思路一断就乱** | 数学、逻辑、英语、写作 | 路径要支持"断点续学 + 快速召回前情" |
| **只刷题不复盘，错题重复错** | 数学、逻辑、英语 | 诊断 Agent 必须沉淀"为什么错"，优化 Agent 推入滚动复习 |
| **下班脑力枯竭，长文本吃不动** | 逻辑、英语、写作 | 资源要分"低耗模式（图解/短讲）"和"高耗模式（精讲/真题）" |
| **缺少解题套路 / 标准模板** | 数学、逻辑、写作 | 讲解资源要按"套路/模板"打标签，而非仅按知识点 |
| **题型特殊、新手完全摸不透** | 数学（条件充分判断）、逻辑（三大题型） | 诊断测验要覆盖题型感知，而不只是知识点 |

> 这些共性说明：本项目的核心价值 "诊断 — 生成 — 动态调整" 闭环在 **所有在职备考场景** 都有普适性，首期在**数学基础**做深做透后，可平移到逻辑、写作、英语。
