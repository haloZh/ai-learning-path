## 二、下一步计划（W2）

1. **扩展数据模型**：新增 `concepts / resources / questions / interactions / mastery_snapshots` 五张表；
2. **知识点种子数据**：录入管综数学 40–60 个知识点节点 + 先修关系（构成知识图谱骨架）；
3. **诊断 Agent**：学生答诊断测验（~15 题）→ LLM 分析 → 输出每个知识点的掌握度（0–1）并持久化为快照。

**W2 交付**：诊断测验能跑端到端 demo，可演示"学生输入答卷 → 系统输出薄弱点诊断报告"。
